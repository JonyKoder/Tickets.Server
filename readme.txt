С помощью инструмента для работы с базами данных(DBeaver) создаём подключение к postgress с настройками подключения к БД
Клонировать репозиторий с github, в своей среде разработки в качестве запускаевомого проекта назанчить "Tickets.Server.Application"
При запуске база данных будет создана автоматически
Откроется инструмент swagger по адресу localhost-а
Методы api готовы к тестированию
